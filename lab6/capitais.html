Parallax

Faça Scroll Up e Down desta página para ver o efeito scrolling parallax.

Os elementos <div class="paralax"> funcionam como janelas/buracos, vendo-se uma imagem fixa no fundo, que não é arrastada com o scroll. Como funciona? Têm como background uma imagem fixa graças à propriedade background-attachment: fixed.

Em baixo temos outro <div class="paralax">. Ao fazer scroll, faz aparecer a mesma imagem anterior, fixa no fundo.

No selector de classe parallax experimente remover a propriedade "background-attachment: fixed;" verá que, quando faz scroll, a imagem será "arrastada" como uma image classica.

Por baixo desta div está uma div com uma imagem como background-image mas sem a propriedade background-attachment: fixed. Por essa razão ela viaja com o texto.
Por baixo deste div existe uma div com uma imagem como background-image, que está fixa ao fundo graças à propriedade background-attachment: fixed.
Este div existe para mostrar e permitir fazer scroll. Dica: no CSS experimente remover a propriedade background-attachment para "arrastar" a imagem dentro da moldura.
